2. Read the input and store it in the height variable.
3. Check if the height is less than 1 or greater than 8. If it is, show an error message and exit the program.
4. use a loop to loop from 1 to height to print each row of the pyramid.

6. loop through row times and print a hash character each time.
7. Go to the next line at the end of the row.
